# Robot Battle Arena

A simple Android app for building and battling robots.

## Features
- Build robots with customizable parts
- Save and load robots using Room DB
- Validate robot configurations
- Battle simulation

## Setup
1. Clone the repo
2. Open in Android Studio
3. Build and run

## Testing
Run unit tests with `./gradlew test`
Run UI tests with `./gradlew connectedAndroidTest`
